Remote Activity Changes

Version 0.1.2

  • Added automatic updates, using Sparkle framework
  • Fixed a bug which arose when extraneous output or error messages are generated by the access shell (eg ssh)
  • Fixed access shell arguments so that options can be passed to ssh, rsh, etc. This can be used for tunneling, for example

Version 0.1.1

  • Improved ps command support by removing query process and its ancestors from the job list
  • Changed code so that you are no longer required to enter a user name, which is not necessary in some cases (eg xgrid)

Version 0.1

First public alpha release of Remote Activity.
